# Setup for my Mac
This contains all the applications I need to get up and running

## Getting going
1. Install git
```
git
```
2. Install home brew
```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
3. Brew install all applications
```
sudo ./install-apps.sh
```
